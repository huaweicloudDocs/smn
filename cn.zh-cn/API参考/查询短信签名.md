# 查询短信签名<a name="ZH-CN_TOPIC_0064366151"></a>

## 功能介绍<a name="section53501716171922"></a>

-   接口名称

    ListSmsSigns

-   功能描述

    查询短信签名列表。

    >![](public_sys-resources/icon-note.gif) **说明：**   
    >该接口暂时仅提供给企业用户使用。
