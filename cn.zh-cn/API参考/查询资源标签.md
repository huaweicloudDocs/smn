# 查询资源标签<a name="ZH-CN_TOPIC_0105885511"></a>

## 功能介绍<a name="section199791258152011"></a>

-   接口名称

    ListResourceTags


-   功能描述

    查询指定实例的标签信息。

    标签管理服务需要使用该接口查询指定实例的全部标签数据。


## URI<a name="section49791258162015"></a>

-   URI格式

    GET /v2/\{project\_id\}/\{resource\_type\}/\{resource\_id\}/tags
