# 批量发送推广类短信<a name="ZH-CN_TOPIC_0094175384"></a>

## 功能介绍<a name="section772195095418"></a>

-   接口名称

    PromotionSmsPublish

-   功能描述

    批量发送推广短信给指定手机号码数组


>![](public_sys-resources/icon-note.gif) **说明：**   
>该接口暂时仅提供给企业用户使用。  

## URI<a name="section272650175410"></a>

-   URI格式

    POST /v2/\{project\_id\}/notifications/sms/promotion
