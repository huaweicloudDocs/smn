# 查询消息模板详情<a name="ZH-CN_TOPIC_0036016767"></a>

## 功能介绍<a name="section17649112"></a>

-   接口名称

    QueryMessageTemplateDetail


-   功能描述

    查询模板详情，包括模板内容。


## URI<a name="section24624288"></a>

-   URI格式

    GET /v2/\{project\_id\}/notifications/message\_template/\{message\_template\_id\}
