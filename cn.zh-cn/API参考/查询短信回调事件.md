# 查询短信回调事件<a name="ZH-CN_TOPIC_0080305750"></a>

## 功能介绍<a name="section38823609153113"></a>

-   接口名称

    ListSmsEvent

-   功能描述

    查询短信回调事件。


>![](public_sys-resources/icon-note.gif) **说明：**   
>该接口暂时仅提供给企业用户使用。  

## URI<a name="section48816673153113"></a>
