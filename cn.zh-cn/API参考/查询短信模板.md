# 查询短信模板<a name="ZH-CN_TOPIC_0093941446"></a>

## 功能介绍<a name="section1436014147464"></a>

-   接口名称

    ListSmsTemplates

-   功能描述

    查询短信模板列表。


>![](public_sys-resources/icon-note.gif) **说明：**   
>该接口暂时仅提供给企业用户使用。  

## URI<a name="section4376414124619"></a>

-   URI格式

    GET /v2/\{project\_id\}/notifications/sms\_template
