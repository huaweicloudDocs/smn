# 添加资源标签<a name="ZH-CN_TOPIC_0105885509"></a>

## 功能介绍<a name="section6275354111818"></a>

-   接口名称

    CreateResourceTag


-   功能描述

    一个资源上最多有10个标签。

    此接口为幂等接口：创建时，如果创建的标签已经存在（key相同），则覆盖。
