# 更新消息模板<a name="ZH-CN_TOPIC_0036016764"></a>

## 功能介绍<a name="section28165387"></a>

-   接口名称

    UpdateMessageTemplate


-   功能描述

    修改消息模板的内容。


## URI<a name="section52161898"></a>

-   URI格式

    PUT /v2/\{project\_id\}/notifications/message\_template/\{message\_template\_id\}
